# Alankara
**Alankara** is a premium jewelry website blending tradition with modern elegance. Built using **HTML5** for structure, **CSS3 &amp; Bootstrap** for responsive styling, and **JavaScript** for dynamic features like cart, modals, and interactivity. The tech stack ensures a fast, user-friendly, and visually rich shopping experience across all devices.
